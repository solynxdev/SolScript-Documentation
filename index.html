What is SolScript?

SolScript is a modern, statically typed, compiled systems programming language built around three core ideas: performance, simplicity, and consistency.

Inspired by C++, Rust, Luau, and Python, SolScript provides familiar syntax, modern language features, and native compiled performance while avoiding unnecessary complexity.

Whether you're writing command-line tools, desktop applications, game engines, or low-level libraries, SolScript is designed to scale from your first "Hello, World!" to large production projects.

Why SolScript?

Performance

Native compiled code with optimization comparable to modern C and C++ compilers.

Simplicity

Learn the language quickly without needing to understand massive APIs or unnecessary boilerplate.

Modern Features

Generics, modules, reflection, interfaces, nullable types, exception handling, automatic garbage collection, and more.
